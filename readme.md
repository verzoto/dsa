# CS

A collection of computer science algorithms and data structures implemented from scratch.

## Contents

### Sorting

- **array.c** - Basic array operations
- **bubble_sort.c** - Bubble Sort
- **selection_sort.c** - Selection Sort
- **insertion_sort.c** - Insertion Sort
- **merge_sort.c** - Merge Sort

## Building

Compile any file with `gcc`:

```sh
gcc Sorting/bubble_sort.c -o bubble_sort
./bubble_sort
```
